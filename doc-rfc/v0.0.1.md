# Feature

- Feature Name:  `Authority Manager`
- Start Date: 2022-11-06

## Summary

A gRPC authority manager system.

## Motivation

We use this project to learn to implement a gRPC system, and to learn database migration and test, and to learn Jwt to authorization.

### Guide-level explanation
```proto
message User{
  int32 id = 1;
  string name = 2;
  string email = 3;
  string password = 4;
  UserAuth auth = 5;
  UserStatus status = 6;
}

enum UserAuth{
  ADMIN,
  CUSTOMER
}

enum UserStatus{
  ACTIVE,
  FREEZE,
  DROPPED,
}

message CreateUserRequest{
  string name = 1;
  string email = 2;
  string password = 3;
}

message CreateUserResponse{
    int32 id = 1;
}



```
